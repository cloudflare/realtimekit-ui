# rtk-paginated-list

## Properties

| Property | Type | Required | Default | Description |
|----------|------|----------|---------|-------------|
| `autoScroll` | `boolean` | ✅ | - | auto scroll list to bottom |
| `createNodes` | `(data: unknown[])` | ✅ | - | Create nodes |
| `emptyListLabel` | `string` | ✅ | - | label to show when empty |
| `fetchData` | `(timestamp: number, size: number, reversed: boo...` | ✅ | - | Fetch the data |
| `iconPack` | `IconPack` | ❌ | `defaultIconPack` | Icon pack |
| `onNewNode` | `(node: DataNode)` | ✅ | - | On a new node created |
| `onNodeDelete` | `(key: string)` | ✅ | - | On node deleted |
| `onNodeUpdate` | `(key: string, newItem: DataNode)` | ✅ | - | On node updated |
| `pageSize` | `number` | ✅ | - | Page Size |
| `pagesAllowed` | `number` | ✅ | - | Number of pages allowed to be shown |
| `t` | `RtkI18n` | ❌ | `useLanguage()` | Language |

## Usage Examples

### Basic Usage

```html
<rtk-paginated-list></rtk-paginated-list>
```

### With Properties

```html
<rtk-paginated-list
  autoScroll=true
  createNodes=[]
  emptyListLabel="example">
</rtk-paginated-list>
```
