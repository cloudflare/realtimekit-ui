# rtk-channel-selector-view

## Properties

| Property | Type | Required | Default | Description |
|----------|------|----------|---------|-------------|
| `channels` | `{
    id: string` | ✅ | - | Channels |
| `disableSearch` | `boolean` | ✅ | - | Disables search bar (default = false) |
| `hideAvatar` | `boolean` | ✅ | - | Hides avatar (default = false) |
| `iconPack` | `IconPack1` | ❌ | `defaultIconPack` | Icon Pack |
| `selectedChannelId` | `string` | ✅ | - | Selected channel id |
| `t` | `RtkI18n1` | ❌ | `useLanguage()` | Language |
| `viewAs` | `'dropdown' | 'list'` | ✅ | - | Render as dropdown or list (default = list) |

## Usage Examples

### Basic Usage

```html
<rtk-channel-selector-view></rtk-channel-selector-view>
```

### With Properties

```html
<rtk-channel-selector-view
  channels="example"
  disableSearch=true
  hideAvatar=true>
</rtk-channel-selector-view>
```
