# rtk-channel-details

## Properties

| Property | Type | Required | Default | Description |
|----------|------|----------|---------|-------------|
| `channel` | `ChatChannel` | ✅ | - | Channel object |
| `iconPack` | `IconPack1` | ❌ | `defaultIconPack` | Icon pack |
| `members` | `RTKBasicParticipant[]` | ✅ | - | List of channel members |
| `t` | `RtkI18n1` | ❌ | `useLanguage()` | Language |

## Usage Examples

### Basic Usage

```html
<rtk-channel-details></rtk-channel-details>
```

### With Properties

```html
<rtk-channel-details
  channel=chatchannel
  members=[]>
</rtk-channel-details>
```
