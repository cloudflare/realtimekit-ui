# rtk-spinner

A component which shows an animating spinner.

## Properties

| Property | Type | Required | Default | Description |
|----------|------|----------|---------|-------------|
| `iconPack` | `IconPack` | ❌ | `defaultIconPack` | Icon pack |
| `size` | `Size1` | ✅ | - | Size |

## Usage Examples

### Basic Usage

```html
<rtk-spinner></rtk-spinner>
```

### With Properties

```html
<rtk-spinner
  size="md">
</rtk-spinner>
```
