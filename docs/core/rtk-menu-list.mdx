# rtk-menu-list

A menu list component.

## Properties

| Property | Type | Required | Default | Description |
|----------|------|----------|---------|-------------|
| `iconPack` | `IconPack` | ❌ | `defaultIconPack` | Icon pack |
| `t` | `RtkI18n` | ❌ | `useLanguage()` | Language |

## Usage Examples

### Basic Usage

```html
<rtk-menu-list></rtk-menu-list>
```

### With Properties

```html
<rtk-menu-list
  iconPack={'defaultIconPack'}
  t=rtki18n>
</rtk-menu-list>
```
