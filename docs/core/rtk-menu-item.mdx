# rtk-menu-item

A menu item component.

## Properties

| Property | Type | Required | Default | Description |
|----------|------|----------|---------|-------------|
| `iconPack` | `IconPack` | ❌ | `defaultIconPack` | Icon pack |
| `size` | `Size` | ✅ | - | Size |
| `t` | `RtkI18n` | ❌ | `useLanguage()` | Language |

## Usage Examples

### Basic Usage

```html
<rtk-menu-item></rtk-menu-item>
```

### With Properties

```html
<rtk-menu-item
  size="md">
</rtk-menu-item>
```
