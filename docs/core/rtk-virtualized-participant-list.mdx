# rtk-virtualized-participant-list

## Properties

| Property | Type | Required | Default | Description |
|----------|------|----------|---------|-------------|
| `bufferedItemsCount` | `number` | Ôťů | - | Buffer items to render before and after the visible area |
| `emptyListElement` | `HTMLElement` | Ôťů | - | Element to render if list is empty |
| `itemHeight` | `number` | Ôťů | - | Height of each item in pixels (assumed fixed) |
| `items` | `Peer1[]` | Ôťů | - | Items to be virtualized |
| `renderItem` | `(item: Peer1, index: number)` | Ôťů | - | Function to render each item |

## Usage Examples

### Basic Usage

```html
<rtk-virtualized-participant-list></rtk-virtualized-participant-list>
```

### With Properties

```html
<rtk-virtualized-participant-list
  bufferedItemsCount=42
  emptyListElement=htmlelement
  itemHeight=42>
</rtk-virtualized-participant-list>
```
