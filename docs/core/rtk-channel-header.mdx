# rtk-channel-header

## Properties

| Property | Type | Required | Default | Description |
|----------|------|----------|---------|-------------|
| `channel` | `ChatChannel` | ✅ | - | Channel object |
| `iconPack` | `IconPack1` | ❌ | `defaultIconPack` | Icon pack |
| `meeting` | `Meeting` | ✅ | - | Meeting object |
| `showBackButton` | `boolean` | ✅ | - | Show back button |
| `t` | `RtkI18n1` | ❌ | `useLanguage()` | Language |

## Usage Examples

### Basic Usage

```html
<rtk-channel-header></rtk-channel-header>
```

### With Properties

```html
<rtk-channel-header
  channel=chatchannel
  meeting={meeting}
  showBackButton=true>
</rtk-channel-header>
```
