# rtk-chat

Fully featured chat component with image & file upload, emoji picker and auto-scroll.

## Properties

| Property | Type | Required | Default | Description |
|----------|------|----------|---------|-------------|
| `config` | `UIConfig1` | ❌ | `createDefaultConfig()` | Config |
| `disablePrivateChat` | `boolean` | ✅ | - | disables private chat |
| `displayFilter` | `ChatFilter` | ✅ | - | @deprecated Beta API, will change in future A filter function for messages to be displayed |
| `iconPack` | `IconPack` | ❌ | `defaultIconPack` | Icon pack |
| `meeting` | `Meeting` | ✅ | - | Meeting object |
| `privatePresetFilter` | `String[]` | ✅ | - | @deprecated Beta API, will change in future List of target presets allowed as private chat recipient |
| `size` | `Size` | ✅ | - | Size |
| `t` | `RtkI18n` | ❌ | `useLanguage()` | Language |

## Usage Examples

### Setup

```typescript
// app.module.ts
import { CUSTOM_ELEMENTS_SCHEMA } from '@angular/core';
import { defineCustomElements } from '@cloudflare/realtimekit-ui/loader';

defineCustomElements();

@NgModule({
  schemas: [CUSTOM_ELEMENTS_SCHEMA]
})
export class AppModule { }
```

### Basic Usage

```html
<!-- component.html -->
<rtk-chat></rtk-chat>
```

### With Properties

```html
<!-- component.html -->
<rtk-chat
  [disablePrivateChat]=true
  [displayFilter]=chatfilter
  [meeting]={meeting}>
</rtk-chat>
```
