# rtk-transcript

A component which shows a transcript.
 You need to remove the element after you receive the
 `rtkTranscriptDismiss` event.

## Properties

| Property | Type | Required | Default | Description |
|----------|------|----------|---------|-------------|
| `t` | `RtkI18n` | ❌ | `useLanguage()` | Language |
| `transcript` | `Transcript & { renderedId?: string }` | ❌ | - | Message |

## Usage Examples

### Setup

```typescript
// app.module.ts
import { CUSTOM_ELEMENTS_SCHEMA } from '@angular/core';
import { defineCustomElements } from '@cloudflare/realtimekit-ui/loader';

defineCustomElements();

@NgModule({
  schemas: [CUSTOM_ELEMENTS_SCHEMA]
})
export class AppModule { }
```

### Basic Usage

```html
<!-- component.html -->
<rtk-transcript></rtk-transcript>
```

### With Properties

```html
<!-- component.html -->
<rtk-transcript
  [t]=rtki18n
  [transcript]="example">
</rtk-transcript>
```
