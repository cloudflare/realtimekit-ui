# RtkChatMessagesUiPaginated

## Properties

| Property | Type | Required | Default | Description |
|----------|------|----------|---------|-------------|
| `iconPack` | `IconPack` | ❌ | `defaultIconPack` | Icon pack |
| `leftAlign` | `boolean` | ✅ | - | Whether to align chat bubbles to the left |
| `meeting` | `Meeting` | ✅ | - | Meeting object |
| `size` | `Size` | ✅ | - | Size |
| `t` | `RtkI18n` | ❌ | `useLanguage()` | Language |

## Usage Examples

### Basic Usage

```tsx
import { RtkChatMessagesUiPaginated } from '@cloudflare/realtimekit-ui/react';

function MyComponent() {
  return <RtkChatMessagesUiPaginated />;
}
```

### With Properties

```tsx
import { RtkChatMessagesUiPaginated } from '@cloudflare/realtimekit-ui/react';

function MyComponent() {
  return (
    <RtkChatMessagesUiPaginated
      leftAlign={true}
      meeting={meeting}
      size="md"
    />
  );
}
```
