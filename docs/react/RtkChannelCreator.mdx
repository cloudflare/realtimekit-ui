# RtkChannelCreator

## Properties

| Property | Type | Required | Default | Description |
|----------|------|----------|---------|-------------|
| `iconPack` | `IconPack1` | ❌ | `defaultIconPack` | Icon pack |
| `meeting` | `Meeting` | ✅ | - | Meeting object |
| `t` | `RtkI18n1` | ❌ | `useLanguage()` | Language |

## Usage Examples

### Basic Usage

```tsx
import { RtkChannelCreator } from '@cloudflare/realtimekit-ui/react';

function MyComponent() {
  return <RtkChannelCreator />;
}
```

### With Properties

```tsx
import { RtkChannelCreator } from '@cloudflare/realtimekit-ui/react';

function MyComponent() {
  return (
    <RtkChannelCreator
      meeting={meeting}
    />
  );
}
```
