# RtkChatMessagesUi

## Properties

| Property | Type | Required | Default | Description |
|----------|------|----------|---------|-------------|
| `canPinMessages` | `boolean` | ✅ | - | Can current user pin/unpin messages |
| `iconPack` | `IconPack1` | ❌ | `defaultIconPack` | Icon pack |
| `messages` | `Chat[]` | ✅ | - | Chat Messages |
| `selectedGroup` | `string` | ✅ | - | Selected group key |
| `selfUserId` | `string` | ✅ | - | User ID of self user |
| `size` | `Size1` | ✅ | - | Size |
| `t` | `RtkI18n` | ❌ | `useLanguage()` | Language |

## Usage Examples

### Basic Usage

```tsx
import { RtkChatMessagesUi } from '@cloudflare/realtimekit-ui/react';

function MyComponent() {
  return <RtkChatMessagesUi />;
}
```

### With Properties

```tsx
import { RtkChatMessagesUi } from '@cloudflare/realtimekit-ui/react';

function MyComponent() {
  return (
    <RtkChatMessagesUi
      canPinMessages={true}
      messages={[]}
      selectedGroup="example"
    />
  );
}
```
