# RtkMenuItem

A menu item component.

## Properties

| Property | Type | Required | Default | Description |
|----------|------|----------|---------|-------------|
| `iconPack` | `IconPack` | ❌ | `defaultIconPack` | Icon pack |
| `size` | `Size` | ✅ | - | Size |
| `t` | `RtkI18n` | ❌ | `useLanguage()` | Language |

## Usage Examples

### Basic Usage

```tsx
import { RtkMenuItem } from '@cloudflare/realtimekit-ui/react';

function MyComponent() {
  return <RtkMenuItem />;
}
```

### With Properties

```tsx
import { RtkMenuItem } from '@cloudflare/realtimekit-ui/react';

function MyComponent() {
  return (
    <RtkMenuItem
      size="md"
    />
  );
}
```
