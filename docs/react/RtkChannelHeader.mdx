# RtkChannelHeader

## Properties

| Property | Type | Required | Default | Description |
|----------|------|----------|---------|-------------|
| `channel` | `ChatChannel` | ✅ | - | Channel object |
| `iconPack` | `IconPack1` | ❌ | `defaultIconPack` | Icon pack |
| `meeting` | `Meeting` | ✅ | - | Meeting object |
| `showBackButton` | `boolean` | ✅ | - | Show back button |
| `t` | `RtkI18n1` | ❌ | `useLanguage()` | Language |

## Usage Examples

### Basic Usage

```tsx
import { RtkChannelHeader } from '@cloudflare/realtimekit-ui/react';

function MyComponent() {
  return <RtkChannelHeader />;
}
```

### With Properties

```tsx

function MyComponent() {
  return (
    <RtkChannelHeader
```
