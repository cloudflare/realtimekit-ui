# RtkDraftAttachmentView

A component which renders the draft attachment to send

## Properties

| Property | Type | Required | Default | Description |
|----------|------|----------|---------|-------------|
| `attachment` | `{
    type: 'image' | 'file'` | ✅ | - | Attachment to display |
| `iconPack` | `IconPack1` | ❌ | `defaultIconPack` | Icon pack |
| `t` | `RtkI18n1` | ❌ | `useLanguage()` | Language |

## Usage Examples

### Basic Usage

```tsx
import { RtkDraftAttachmentView } from '@cloudflare/realtimekit-ui/react';

function MyComponent() {
  return <RtkDraftAttachmentView />;
}
```

### With Properties

```tsx
import { RtkDraftAttachmentView } from '@cloudflare/realtimekit-ui/react';

function MyComponent() {
  return (
    <RtkDraftAttachmentView
      attachment={{}}
    />
  );
}
```
